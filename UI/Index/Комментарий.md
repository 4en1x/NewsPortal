В итоговой версии кнопки "Раскрыть новость" скорее всего не будет и переход к самой новости будет осуществляться по нажатию на картинку/заголовок. Она была добавлена для соответствия ТЗ
Поисковая панель также недоработана, т.к. пока нельхя использовать js. Впоследствии при наборе тега он будет добавляться в некий список(и отображаться,с кнопочкой удалить), а поле для ввода будет очищаться.
Переход на панель поиска происходит по наведении на соответствующую кнопку.
